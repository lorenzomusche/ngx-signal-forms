## Must-read files

Prima di eseguire qualsiasi task in questo workspace:
- Leggi e applica le regole contenute in `.github/*.md`.
- Considera questi file come *fonte autorevole* per:
  - stile di codice
  - convenzioni di commit
  - regole di test e QA

## Execution policy

Quando ricevi un prompt:
- Passo 1: sintetizza in poche bullet le regole rilevanti trovate in `.github`.
- Passo 2: esegui il task richiesto applicando fedelmente tali regole.
- Passo 3: spiega brevemente quali regole hai usato.
